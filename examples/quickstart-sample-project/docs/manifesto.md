# Quickstart Sample Manifesto

This project exists to teach the smallest useful Sha8al Command Center workflow.

## Principles

- Start with the public `generic` profile.
- Keep `docs/roadmap.md` as the task source of truth.
- Keep the example small enough to understand in one sitting.
- Make the Codex and Claude orchestration path visible.
- Show where specialist agents join the flow without requiring a large codebase.

## Scope

- bootstrap from a local sample project
- generate a tracker from markdown
- open the desktop app against that tracker
- run the basic command flow with agent handoffs

## Non-Goals

- TalkStore-only setup
- external services
- a production-sized roadmap
